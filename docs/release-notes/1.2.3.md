# Money Leak 1.2.3 Release Notes

Money Leak 1.2.3 sharpens the core local tracking experience and finalizes this release for TestFlight distribution.

- Track expenses locally with a simple, focused flow.
- Mark purchases as leaks to separate impulse spending from normal spend.
- Review spending and leak patterns in Analytics.
- Share Shame Cards for leak moments.
- Use reminders to stay consistent with check-ins.
- Import and export your data with CSV.
- Open Support and Legal links from Settings.
- Includes release and TestFlight readiness improvements.
