Remote Contract
- Added migration:
  supabase/migrations/20260615000000_create_remote_settings.sql
- New table:
  public.remote_settings
- Columns:
  user_id uuid references auth.users(id) on delete cascade
  key text not null
  value text not null
  updated_at timestamptz not null
  schema_version integer not null default 1
  source_device_id text null
- Primary key:
  (user_id, key)
- Allowed keys:
  currency
  language
- RLS enabled.
- public/anon access revoked.
- authenticated users can select/insert/update/delete only their own rows.
- Sync adapters upsert settings with onConflict user_id,key, so repeated backup/sync does not duplicate rows.
