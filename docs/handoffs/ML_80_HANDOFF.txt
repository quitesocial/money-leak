ML-80 Clean App Icon Handoff

Root cause
- Expo resolved expo.icon, ios.icon, and the Android adaptive foreground to
  assets/images/icon.png.
- That icon was a 1024x1024 RGBA PNG with transparency, baked rounded corners,
  edge shadowing, and light droplet strokes/halo baked into the raster.
- iOS applies its own native icon mask, so the baked mask and halo remained
  visible on the Home Screen and App Library.

Implementation notes
- Rebuilt assets/images/icon.png as a clean 1024x1024 opaque RGB PNG.
- The new iOS/global icon uses a full-square #01070F background and a centered
  Money Leak droplet derived from the clean droplet alpha in
  assets/images/splash-logo.png.
- The new iOS/global icon has no alpha, no baked rounded corners, no icon
  shadow, and no white/light droplet halo or stroke.
- Added assets/images/adaptive-icon.png as a separate transparent Android
  adaptive foreground.
- Updated Android adaptive backgroundColor to #01070F.
- Left the expo-splash-screen config unchanged; it still uses
  assets/images/splash-logo.png on #E5E5EA.
