Date: 2026-06-16
Project: Money Leak
Version after task: 1.27.0

Summary

ML-91 is implemented and manually tested. Analytics & Leaks now has a Figma-aligned Overview section above the Transactions ledger. The existing ledger remains read-only and keeps the same filter semantics.

Key product behavior implemented

- Overview totals are scoped to the selected Analytics period: Today, Week, Month, or Custom.
- Income total is the sum of active balance entries in the selected period.
- Expenses total is the sum of active expense transactions in the selected period.
- Leaks total is the sum of active expense transactions with isLeak === true in the selected period.
- Deleted/tombstoned balance entries and transactions are ignored.
- Invalid or non-finite amounts are sanitized to 0.
- Overview is intentionally independent from Analytics ledger filters. Filters still affect only Transactions rows.
- Home period state is not mutated.
- Balance additions remain a separate domain and are not encoded as Transactions.
- Transaction.category still stores the stable category ID only.
- Transaction CSV v1 remains unchanged: id,amount,category,isLeak,leakReason,note,createdAt.
- Currency remains display-only; no FX conversion or per-entry currency fields were added.
- System labels are localized; custom category names, custom balance type names, notes, and user-generated values are not translated.

Chart implementation details

- Uses react-native-svg, installed with npx expo install react-native-svg.
- The chart is dynamic and rebuilds from overview.chartParts:
  - yellow = expenses excluding leaks, calculated as expenses - leaks
  - green = income
  - red = leaks
- Chart shape follows Figma node 272:4095:
  - filled annular sectors, not strokeLinecap round pills
  - outer arc + inner arc + straight radial sides
  - small quadratic corner rounding
  - gap only between different sections
  - no gap when there is only one non-zero segment
- Current chart constants in analytics-screen.tsx:
  - size: 160
  - outer radius: 78
  - inner radius: 39
  - corner radius: 9
  - gap: 4.5 degrees
  - start angle: 270 degrees
- Empty data renders a neutral grey ring.
- Single-segment data renders one continuous colored SVG Circle with no artificial break.
- Chart animates on rebuild with a short fade/scale transition using React Native Animated.
